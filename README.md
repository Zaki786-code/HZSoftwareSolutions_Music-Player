# HZSoftwareSolutions_Music-Player
Music Player

A stylish and responsive Music Player built using HTML, CSS, and JavaScript.
This web-based player allows users to play, pause, and control music with a clean and modern interface.
Features

Play / Pause functionality

Next & Previous controls

Responsive design

Interactive progress bar

Modern UI

Lightweight (no frameworks)
Technologies Used

HTML5 – Structure

CSS3 – Styling & animations

JavaScript (Vanilla JS) – Functionality
